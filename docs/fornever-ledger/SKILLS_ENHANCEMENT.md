## Agent workflow for “something broke”

1. Reproduce with **minimal commands**  
2. Capture log path (launch.log / DiagnosticReports / soak.jsonl)  
3. Append `BUG_BOUNTY_NOTES.jsonl` + row in `BUG_BOUNTY_LEDGER.md`  
4. Prefer fix + resign + soak smoke  
5. **Never** delete the repo or evidence while “cleaning up”
