# Memory Glass · what it is growing into

> **AI-findable north star** for Grok Build / hotpipe agents / human handoff.  
> **Updated:** 2026-07-18 · tip follows `origin/main`  
> **Not:** generic AI browser (Dia/Comet/Atlas). **Is:** native dual-pane **training + dual-space lab shell**.

---

## One-liner

**Native WKWebView droplet** (tao+wry) with **hot-pipe inject** that turns WebGrid, language geometry, quantum education, and multi-seat mesh into one **glass collaborative instrument** — measure → train → evidence → human share — without Electron bloat or auto-post.

---

## Product thesis (vs market)

| Market AI browsers | Memory Glass |
|--------------------|--------------|
| Chat-first / agent-drive-the-web | **Measure-first** · BPS, strain, path, gates |
| Tab + sidebar copilot | **Dual glass** main + inspect + floats |
| Cloud memory product | Local packs · leaderboard · human X draft |
| Chromium feature race | **WK speed floor** + 1s JS inject loop |

**Category lead:** WebGrid as first-class instrument · path/contrail dual-space · KBatch body · evidence loop.  
**Deliberately weak:** everyday inbox AI, multi-site booking agents, extension store.
